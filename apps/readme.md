## To start an application, run

```
cd public && php -S 127.0.0.1:8000

OR

php -S 127.0.0.1:8000 -t APP-NAME/public

To enable debugbar

APP_ENV_IS_DEVELOPMENT=true php -S 127.0.0.1:8000 -t APP-NAME/public
```
