# Markdown

## Headings

# One is heading 1
## Two is heading 2
### Three is heading 3
#### Four is heading 4


## Text style

Bold - **This is bold**
Italic - *This is italic*
Bold and italic - ***This is bold and italic***
Strikethrough - ~~This is strikethrough~~


## Link

(/?id=1&uri=post)
(http://example.org)
[Link Text](http://example.org)


## List

### Unordered list

* Item 1
	* Item 1.1
	* Item 1.2
		- Item 1.2.1
		- Item 1.2.2
* Item 2
* Item 3

### Ordered list

1. Item 1
2. Item 2
3. Item 3

### Task lists

- [x] Task list item checked
- [X] Task list item checked and striked
- [!] Task list item striked unchecked
- [ ] Task list item unchecked
